## 最终项目要求

### 项目规格

#### 基本要求

1. **技术栈要求**：
   - 必须使用本课程学过的至少3种技术
   - 包含前端（HTML/CSS/JavaScript）
   - 包含后端（Flask或其他框架）
   - 包含数据库（SQLite或其他）

2. **功能要求**：
   - 用户认证系统
   - 数据增删改查（CRUD）
   - 响应式设计
   - 基本的API接口

3. **文档要求**：
   - 项目说明文档
   - 技术架构图
   - API文档
   - 部署说明

## 快速开始

### 环境要求

- Python 3.8+
- pip
- 虚拟环境 (推荐)

### 安装步骤

1. 克隆仓库
```bash
git clone https://github.com/yourusername/projectname.git
cd projectname
```

2. 创建虚拟环境
```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
```

3. 安装依赖
```bash
pip install -r requirements.txt
```

4. 初始化数据库
```bash
flask db upgrade
```

5. 运行应用
```bash
flask run
```

应用将在 http://localhost:5000 上运行。

## 部署指南

### Heroku部署

1. 安装Heroku CLI
2. 登录Heroku: `heroku login`
3. 创建应用: `heroku create`
4. 推送代码: `git push heroku main`
5. 初始化数据库: `heroku run flask db upgrade`

### Docker部署

```bash
# 构建镜像
docker build -t projectname .

# 运行容器
docker run -p 5000:5000 projectname
```

## 测试

运行测试套件:

```bash
python -m pytest tests/
```

生成覆盖率报告:

```bash
python -m pytest --cov=app tests/
```
