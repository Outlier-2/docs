# CMU 15-445 数据库系统课程概览

CMU 15-445：卡内基梅隆大学数据库系统课程，被誉为世界上最好的数据库系统课程

## 课程简介

CMU 15-445是卡内基梅隆大学的研究生级别数据库系统课程，涵盖从关系模型到分布式数据库的完整学习路径。通过26讲的系统学习，你将掌握数据库系统的核心原理、实现技术和前沿发展。

## 课程结构

<a id="course-structure"></a>

### 完整学习路径
```
Lecture 1-2  → 关系模型与SQL基础
Lecture 3-6  → 数据库存储与内存管理
Lecture 7-9  → 索引结构与并发控制
Lecture 10-14 → 查询处理与优化
Lecture 15-18 → 并发控制算法
Lecture 19-20 → 日志与恢复
Lecture 21-23 → 分布式数据库系统
Lecture 24-26 → 应用与综合
```

### 讲座概览

| 讲座 | 主题 | 核心内容 | 关键概念 |
|------|------|----------|----------|
| Lecture 1 | 课程概述与关系模型 | 数据库系统介绍、关系代数 | 关系模型、SQL基础、数据完整性 |
| Lecture 2 | 高级SQL | 复杂查询、窗口函数、递归查询 | SQL优化、查询计划、性能调优 |
| Lecture 3 | 数据库存储 I | 磁盘存储、文件组织、缓冲池 | 存储层次、页面管理、I/O优化 |
| Lecture 4 | 数据库存储 II | 堆文件、页面格式、记录管理 | 数据布局、存储开销、访问模式 |
| Lecture 5 | 存储模型与压缩 | 列式存储、压缩算法 | 存储效率、查询性能、压缩技术 |
| Lecture 6 | 内存管理 | 缓冲池管理、替换策略 | LRU、时钟算法、内存优化 |
| Lecture 7 | 哈希表 | 静态/动态哈希、扩展哈希 | 哈希函数、冲突解决、索引结构 |
| Lecture 8 | 树索引 | B+树、索引优化 | 平衡树、索引选择、范围查询 |
| Lecture 9 | 索引并发控制 | B+树并发、锁协议 | 并发访问、死锁处理、性能优化 |
| Lecture 10 | 排序与聚合算法 | 外部排序、分组操作 | 排序算法、聚合函数、内存管理 |
| Lecture 11 | 连接算法 | 嵌套循环、哈希连接、归并连接 | 查询优化、执行策略、成本估算 |
| Lecture 12 | 查询执行 I | 查询处理框架、执行引擎 | 查询计划、执行模型、流水线处理 |
| Lecture 13 | 查询执行 II | 表达式求值、物化 | 查求值、内存管理、执行优化 |
| Lecture 14 | 查询计划与优化 | 查询优化器、统计信息 | 优化策略、成本模型、计划选择 |
| Lecture 15 | 并发控制理论 | 并发问题、隔离级别 | 一致性、隔离性、并发理论 |
| Lecture 16 | 两阶段锁定并发控制 | 2PL协议、死锁处理 | 锁机制、死锁预防、并发控制 |
| Lecture 17 | 时间戳排序并发控制 | 时间戳协议、冲突处理 | 时间戳管理、可串行化、性能分析 |
| Lecture 18 | 多版本并发控制 | MVCC、快照隔离 | 版本管理、并发性能、实现策略 |
| Lecture 19 | 数据库日志 | 日志协议、恢复基础 | 日志记录、检查点、恢复算法 |
| Lecture 20 | 数据库恢复 | ARIES算法、崩溃恢复 | 恢复机制、事务管理、数据一致性 |
| Lecture 21 | 分布式数据库简介 | 分布式系统概念、数据分区 | 分布式架构、数据分布、一致性模型 |
| Lecture 22 | 分布式OLTP系统 | 事务处理、复制协议 | 分布式事务、复制策略、一致性保证 |
| Lecture 23 | 分布式OLAP系统 | 数据仓库、分析查询 | 大数据分析、查询优化、并行处理 |
| Lecture 24 | 数据库内应用逻辑 | 存储过程、触发器 | 应用集成、性能优化、安全性 |
| Lecture 25 | 嘉宾演讲：Snowflake | 现代数据仓库架构 | 云数据库、架构设计、实践案例 |
| Lecture 26 | 最终复习 | 课程总结、考试准备 | 核心概念回顾、重点梳理 |

## 学习目标和成果

### 核心收获
- 深入理解数据库系统的内部实现原理
- 掌握查询处理、优化和并发控制的核心技术
- 具备分布式数据库系统的理论基础
- 获得实现完整数据库系统的实践经验

### 学习进度
- 理解每个讲座的核心概念和算法
- 完成配套的编程项目和实验
- 阅读相关论文和扩展材料
- 实现一个微型数据库系统
