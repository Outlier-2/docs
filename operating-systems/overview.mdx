# 操作系统原理

南京大学操作系统2025：AI时代下的操作系统设计与实现

## 课程简介

本课程基于南京大学操作系统2025年春季学期课程，聚焦AI时代下的操作系统设计与实现。课程从应用、硬件、数学多视角出发，系统讲解操作系统的核心概念和实现原理。

### 教学理念

"AI时代的操作系统课" - 结合现代计算需求，强调虚拟化、并发、持久化及安全，培养学生构建复杂系统的能力。

### 课程特色

- **AI时代视角**：探讨AI对操作系统设计的影响和需求
- **多维度教学**：应用、硬件、数学三个视角融合
- **理论与实践结合**：每个模块都配有配套实验
- **现代操作系统**：关注当前操作系统发展的前沿趋势
