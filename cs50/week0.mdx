## 核心概念

### 计算思维与问题解决

**计算机科学本质上是问题解决的学科**。每个问题都有：

- **输入**：需要处理的数据
- **输出**：期望得到的结果
- **解决过程**：从输入到输出的步骤

![输入输出处理图示](https://cs50.harvard.edu/x/notes/0/cs50Week0Slide38.png)

### 数据表示系统

计算机只理解0和1，但通过这些简单数字可以表示一切：

#### 二进制系统

**为什么使用二进制？**
- **物理实现简单**：电路的开/关状态
- **可靠性高**：只有两种状态，抗干扰能力强
- **数学基础**：基于布尔代数

**二进制表示示例**：
```text
13 (十进制) = 8 + 4 + 0 + 1 = 1101 (二进制)
42 (十进制) = 32 + 8 + 2 = 101010 (二进制)
```

#### ASCII和Unicode

**ASCII**：用7位二进制表示英文字符
- A = 65 = 1000001
- a = 97 = 1100001
- 0 = 48 = 0110000

Unicode可以表示全球各种语言的字符，包括表情符号等特殊字符。

### 算法与抽象化

#### 什么是算法？

**算法是解决问题的精确步骤**，例如找最大值的算法：

```pseudo
function findMax(a, b, c):
    if a > b and a > c:
        return a
    else if b > a and b > c:
        return b
    else:
        return c
```

#### 抽象化的重要性

**抽象化是隐藏复杂细节的过程**：
- 函数：隐藏具体实现，只暴露接口
- 变量：用名称表示内存中的值
- 控制结构：简化重复和条件逻辑

## 算法效率

### 运行时间分析

好的算法不仅要正确，还要高效：

- **线性时间 O(n)**：运行时间与输入大小成正比
- **对数时间 O(log n)**：每次都将问题规模减半
- **常数时间 O(1)**：运行时间与输入大小无关

![算法效率对比图](https://cs50.harvard.edu/x/notes/0/cs50Week0Slide141.png)

上图比较了不同搜索算法的效率：线性搜索（O(n)）、每次减半的搜索（O(n/2)）和二分查找（O(log n)）。可以看到算法效率对程序性能的巨大影响。

### 二分查找示例

```pseudo
function binarySearch(array, target):
    left = 0
    right = length(array) - 1

    while left <= right:
        middle = (left + right) / 2
        if array[middle] == target:
            return middle
        else if array[middle] < target:
            left = middle + 1
        else:
            right = middle - 1

    return -1  // 未找到
```

## 代码示例

### 十进制转二进制

```c
#include <stdio.h>

void decimalToBinary(int n) {
    // 处理0的特殊情况
    if (n == 0) {
        printf("0");
        return;
    }

    int binary[32];
    int i = 0;

    // 除2取余法
    while (n > 0) {
        binary[i] = n % 2;
        n = n / 2;
        i++;
    }

    // 逆序输出
    printf("二进制表示: ");
    for (int j = i - 1; j >= 0; j--) {
        printf("%d", binary[j]);
    }
}

int main() {
    int number;
    printf("请输入一个十进制数: ");
    scanf("%d", &number);

    printf("%d 的二进制是: ", number);
    decimalToBinary(number);
    printf("\n");

    return 0;
}
```

### 简单算法实现

```c
#include <stdio.h>

// 找出三个数中的最大值
int findMax(int a, int b, int c) {
    int max = a;

    if (b > max) {
        max = b;
    }

    if (c > max) {
        max = c;
    }

    return max;
}

// 计算数组平均值
double calculateAverage(int array[], int size) {
    int sum = 0;

    for (int i = 0; i < size; i++) {
        sum += array[i];
    }

    return (double)sum / size;
}

int main() {
    // 测试findMax函数
    int num1 = 10, num2 = 25, num3 = 15;
    int maximum = findMax(num1, num2, num3);
    printf("三个数中的最大值是: %d\n", maximum);

    // 测试average函数
    int scores[] = {85, 90, 78, 92, 88};
    int size = sizeof(scores) / sizeof(scores[0]);
    double average = calculateAverage(scores, size);
    printf("平均分是: %.2f\n", average);

    return 0;
}
```

## 练习题与答案

### 1. 二进制转换练习

**问题：** 将以下十进制数转换为二进制

**答案：**
- 25 = 16 + 8 + 0 + 0 + 1 = 11001（二进制）
- 42 = 32 + 0 + 8 + 0 + 2 + 0 = 101010（二进制）
- 128 = 10000000（二进制）
- 255 = 128 + 64 + 32 + 16 + 8 + 4 + 2 + 1 = 11111111（二进制）

### 2. ASCII编码

**问题：** 查找以下字符的ASCII值

**答案：**
- 'A' = 65（十进制）= 01000001（二进制）
- 'z' = 122（十进制）= 01111010（二进制）
- '9' = 57（十进制）= 00111001（二进制）
- '@' = 64（十进制）= 01000000（二进制）

### 3. 算法设计

**问题：** 设计以下算法

**答案：**

**判断偶数算法：**
```pseudo
function isEven(n):
    if n % 2 == 0:
        return true
    else:
        return false
```

**计算阶乘算法：**
```pseudo
function factorial(n):
    if n <= 1:
        return 1
    else:
        return n * factorial(n-1)
```

**找数组最小值算法：**
```pseudo
function findMin(array):
    min = array[0]
    for i from 1 to length(array)-1:
        if array[i] < min:
            min = array[i]
    return min
```

### 5. 思考题

**问题：** 为什么计算机使用二进制而不是十进制？

**答案：** 因为二进制在物理实现上更简单、可靠。电路只有开/关两种状态，抗干扰能力强，且基于布尔代数的数学基础。

**问题：** 抽象化如何帮助我们更好地理解复杂系统？

**答案：** 抽象化隐藏了不必要的细节，让我们专注于核心功能和逻辑。就像开车时不需要了解发动机的具体工作原理一样，抽象化让我们能更高效地使用和构建复杂系统。

**问题：** 算法效率为什么重要？

**答案：** 算法效率直接影响程序的运行时间和资源消耗。在大数据量情况下，效率低的算法可能导致程序运行时间过长甚至无法完成，而高效的算法能节省计算资源和时间成本。
