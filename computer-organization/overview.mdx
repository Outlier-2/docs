# 计算机组成原理

MIT 6.004 Computation Structures：从门电路到并行处理

## 课程简介

本课程基于MIT 6.004 Computation Structures课程，系统讲解数字系统架构，从基本的门电路到复杂的并行处理系统。课程采用自底向上的方法，逐步构建完整的计算机系统。

### 教学理念

"从门电路到并行处理" - 通过实践性学习，理解数字系统的设计原理和实现方法。

### 课程特色

- **系统化学习路径**：从基础逻辑门到复杂计算机系统
- **理论与实践结合**：每个概念都配有相应的实验项目
- **现代计算架构**：涵盖最新的处理器设计和并行计算技术
- **工程思维培养**：强调系统设计和性能优化

## 课程结构

### 第一部分：数字逻辑基础 (第1-4讲)
- **第1讲：计算系统基础** - 计算系统的层次结构、信息表示
- **第2讲：数字电路基础** - CMOS电路、布尔代数、逻辑门
- **第3讲：组合逻辑电路** - 组合逻辑设计、优化方法
- **第4讲：时序逻辑电路** - 触发器、寄存器、状态机

### 第二部分：处理器设计 (第5-7讲)
- **第5讲：指令集架构** - ISA设计、寻址模式、指令格式
- **第6讲：处理器设计** - 数据通路、控制单元、单周期处理器
- **第7讲：流水线技术** - 流水线设计、冒险处理、性能优化

### 第三部分：存储系统 (第8-9讲)
- **第8讲：内存层次结构** - Cache设计、存储器层次、性能分析
- **第9讲：虚拟内存** - 页表机制、TLB、内存管理

### 第四部分：系统与性能 (第10-12讲)
- **第10讲：I/O系统** - 中断、DMA、总线设计
- **第11讲：并行计算** - 多核架构、GPU计算、分布式系统
- **第12讲：性能优化** - 性能分析、优化技术、现代架构

## 学习目标

完成本课程后，学生将能够：

1. **理解数字系统**：掌握从门电路到完整计算机系统的设计原理
2. **设计处理器**：能够设计简单的处理器和流水线
3. **优化性能**：理解和应用性能优化技术
4. **系统思维**：具备计算机系统的整体设计思维
5. **实践能力**：通过实验项目掌握硬件描述语言和设计工具

## 实验项目

课程包含12个实验项目，每个实验与对应的讲义内容紧密结合：

- **Lab 1**: 数字逻辑设计与仿真
- **Lab 2**: 组合逻辑电路设计
- **Lab 3**: 时序逻辑电路实现
- **Lab 4**: 简单处理器设计
- **Lab 5**: 流水线处理器实现
- **Lab 6**: Cache系统设计
- **Lab 7**: 虚拟内存系统
- **Lab 8**: I/O系统设计
- **Lab 9**: 并行计算实现
- **Lab 10**: 性能优化项目
- **Lab 11**: 综合系统设计
- **Lab 12**: 最终项目

- **编程基础**：C语言或类似编程语言

- **逻辑思维**：基本的逻辑推理能力

## 学习建议

1. **理论与实践并重**：每个概念都要通过实验加深理解
2. **循序渐进**：按照课程结构逐步学习，不要跳跃
3. **工具掌握**：熟练使用硬件描述语言和仿真工具
4. **项目驱动**：通过实验项目巩固理论知识
5. **思维训练**：培养系统化的工程设计思维
